#프로젝트 이름
MiniGameParty


#프로젝트 개요
Unity VR기반의 MetaQuest를 활용한 캐주얼 게임

#주요기능
- 트리거, 인풋 시스템을 이용한 미니게임 구현
- Unity 자체기능인 음성인식을 이용한 영창 게임 구현
- raycast를 활용한 물체인식 및 상호작용 기능구현
- 제한시간 기능
- 싱글톤

#사용기술
- UnityEngine
- Unity VR
- C#
